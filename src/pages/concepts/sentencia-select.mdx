## Ejemplos
### Recuperar todos los campos
```sql
SELECT * FROM Customers
```
### Filtrar solo algunos campos
```sql
SELECT id, name, age FROM Customers
```

### Contar numero de registros
```sql
SELECT COUNT(*) FROM Customers
```
